# CodeGreen — Technology Solutions Corporate Website

A feature-rich multi-page corporate website for **Green Code Technology**, a Saudi Arabia-based tech company headquartered in Al Khobar, offering AI-driven digital solutions, web & mobile development, cybersecurity, and cloud hosting services.

## 🛠️ Tech Stack

- **HTML5 & CSS3** — semantic multi-page structure and custom styling
- **Bootstrap 5.3** — responsive grid, navbar, collapse
- **Font Awesome 6.0** — icon set
- **Vanilla JavaScript** — custom slider, stats counter, scroll behaviors

## 📄 Pages

- `index.html` — Home
- `about.html` — About Us
- `services.html` — Services overview
- `projects.html` — Projects showcase
- `dream.html` — Make Your Dream (project inquiry)
- `careers.html` — Careers / Recruitment
- `help.html` — Help / FAQ
- `contact-us.html` — Contact page

## ✨ Features

- **Custom UI/UX image slider** — dot-navigated auto-advancing figure slider in a phone/frame mockup on the hero
- **Spinning logo** — CSS-animated rotating logo disc inside concentric rings (Vision/Mission center piece)
- **Vision & Mission section** — 3-column layout with Vision text (left), spinning logo (center), Mission text (right)
- **Hosting pricing cards** — 3 styled pricing cards (VPS Virtual Servers from 60 SAR, Private Servers from 30 SAR, Shared Hosting from 25 SAR) with CPU/RAM/DISK/DATA specs per plan
- **Projects showcase section** — project cards grid with descriptions
- **Animated stats counter** — key company numbers with count-up animation
- **Clients/Partners carousel** — client logo slider
- **AR language toggle button** — "AR" pill button in navbar for Arabic switching
- **Contact page** — "Get In Touch" split layout with phone (+966 13 330 8533), email (ceo@greencodet.com), and Al Khobar location + inquiry form (name, email, subject dropdown, message)
- **3-column footer** — About Company links, Services links (AI, Cyber Security, Networks, Hosting, VR/AR, Design), social icons (Instagram, X, Facebook, LinkedIn)
- Fully responsive across all screen sizes
